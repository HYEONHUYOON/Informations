# BFS(Breadth-First Search) 알고리즘

## 개요

너비 우선 탐색(BFS)은 그래프나 트리 구조에서 노드를 탐색하는 알고리즘으로, **가까운 노드부터 먼저 탐색**하는 방식입니다. 주로 최단 경로 문제에 활용됩니다.

## 동작 원리

1. **시작 노드**에서 탐색을 시작합니다
2. 시작 노드의 **인접 노드를 모두 방문**합니다
3. 방문한 인접 노드의 **인접 노드를 차례로 방문**합니다
4. 이 과정을 **목표 노드를 찾거나 모든 노드를 방문할 때까지** 반복합니다

## 구현 방법

BFS는 **큐(Queue)** 자료구조를 사용하여 구현합니다:

```javascript
const getDistance = (start, goal) => {
    const visited = new Set([start])  // 방문한 노드 기록
    let queue = [[start, 0]]          // 탐색할 노드와 거리를 저장하는 큐

    while(queue.length !== 0){
        const [node, distance] = queue.shift()  // 큐에서 노드 꺼내기

        if(node === goal) return distance;      // 목표 노드 발견 시 거리 반환
        
        graph[node].near.forEach((next) => {    // 인접 노드 탐색
            if(!visited.has(next)) {            // 방문하지 않은 노드만 처리
                visited.add(next)               // 방문 표시
                queue.push([next, distance + 1]) // 큐에 추가 (거리 1 증가)
            }
        })
    }
    return -1;  // 목표 노드를 찾지 못한 경우
}
```

## 특징

- **완전성(Completeness)**: 목표 노드가 존재한다면 반드시 찾을 수 있습니다
- **최적성(Optimality)**: 모든 간선의 가중치가 동일할 때 최단 경로를 보장합니다
- **시간 복잡도**: O(V+E) (V: 노드 수, E: 간선 수)
- **공간 복잡도**: O(V)

## 활용 사례

1. **최단 경로 찾기**: 키패드 문제에서 두 키 사이의 최소 이동 거리 계산
2. **네트워크 분석**: 소셜 네트워크에서 친구 관계 분석
3. **웹 크롤링**: 웹 페이지 간의 연결 관계 탐색
4. **퍼즐 게임 풀이**: 미로 찾기, 슬라이딩 퍼즐 등

## 키패드 문제 적용

키패드 문제에서는 BFS를 통해:
- 현재 손가락 위치에서 목표 키까지의 최단 거리를 계산
- 그래프 구조로 표현된 키패드에서 각 키 간의 연결 관계를 탐색
- 두 손가락 중 어느 것이 목표 키에 더 가까운지 판단

BFS는 이처럼 실제 문제에 적용할 때 그래프 구조와 최단 경로를 활용하는 강력한 알고리즘입니다.
